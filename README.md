# Bank Management System (Java)
📌 Developed as part of Java practice and GitHub portfolio building.

## 📌 Project Description
A console-based Bank Management System developed using Core Java. 
This project demonstrates Object-Oriented Programming concepts such as 
encapsulation, constructors, and method implementation.

## 🚀 Features
- Create bank account
- Deposit money
- Withdraw money
- Check account balance
- Input validation handling

## 🛠 Technologies Used
- Java
- OOP Concepts
- IntelliJ IDEA

## 📂 Project Structure
bankmanagement/
 ├── BankAccount.java
 └── BankMain.java

## 🎯 Learning Outcome
- Improved understanding of OOP
- Practical implementation of real-world banking logic
- Hands-on experience with Git and GitHub
